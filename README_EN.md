# OfferPilot

An AI interviewer that simulates real technical interviews from major Chinese tech companies.

Flow: role setup -> AI follow-up questions -> real-time scoring -> interview report.

## Stack

- Next.js (App Router + TypeScript)
- Tailwind CSS
- Zustand
- Vercel AI SDK
- Recharts
- Monaco Editor

## Run

```bash
npm install
npm run dev
```

Open: http://localhost:3000

## Current Status

- MVP interview flow complete
- Report generation complete
- Local history storage complete
